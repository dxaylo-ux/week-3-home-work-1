Ин лоиҳа веб-сайти мағозаи мебел бо номи «СТОЛ ПЛЮС» мебошад.
Сайт барои муаррифии маҳсулот, хизматрасонӣ ва пешниҳодҳои ширкат сохта шудааст.

Дар қисми аввали сайт менюи навигатсионӣ, ҷустуҷӯи маҳсулот ва баннери асосӣ ҷойгир шудааст.
Истифодабаранда метавонад маҳсулоти лозимаро ҷустуҷӯ намояд ва бо пешниҳодҳои асосӣ шинос шавад.

Қисми «Популярные категории» категорияҳои машҳури мебелро нишон медиҳад, аз ҷумла стулҳо, креслоҳо,
консолҳо ва диванҳо. Барои ҳар як маҳсулот акс, ном ва нарх нишон дода шудааст.

Дар бахши «Новости и предложения» ахбор, мақолаҳо ва пешниҳодҳои махсуси ширкат ҷойгир шудаанд.
Ҳамчунин баннери YouTube мавҷуд аст, ки корбаронро ба канали ширкат роҳнамоӣ мекунад.

Қисми «Отзывы клиентов» фикру мулоҳизаҳои муштариёнро нишон медиҳад,
ки барои баланд бардоштани эътимоди харидорон мусоидат мекунад.

Дар бахши «СТОЛ ПЛЮС» маълумоти мухтасар дар бораи ширкат, таҷрибаи корӣ, сифати маҳсулот,
муносибати инфиродӣ ва нархҳои дастрас пешниҳод шудааст.

Дар қисми Footer маълумотҳои тамос, каталог,
бахши кӯмак ва маълумоти иловагӣ ҷойгир шудаанд.

Лоиҳа бо истифода аз HTML ва Tailwind CSS сохта шуда
, дорои тарҳи муосир ва responsive мебошад, ки дар телефон,
планшет ва компютер дуруст намоиш дода мешавад.
